# Telegram Chat Level System Bot

一個為 Telegram 群組設計的等級系統，透過訊息互動（文字、媒體、貼圖）自動累積經驗並提升等級，增加群組活躍度。

## 核心功能

- **全方位互動計分**：支援文字、圖片、影片、檔案與貼圖。自動排除「指令」與「系統訊息」，確保數據公平。
- **互動排行榜**：新增 `/top` 指令，支援分頁顯示群組等級排行，讓成員隨時查看競爭進度。
- **RPG 等級模型**：採用標準二次公式（`5L^2 + 50L + 10`），隨著等級提升挑戰性隨之增加。
- **高負載穩定性**：優化連線逾時設定並啟用 WAL 模式，即使在網路不穩或大量訊息湧入時也能穩定運作。
- **智慧自動清理**：內建自動刪除機制（預設 60 秒），自動清理指令回應與升級公告，保持群組整潔。
- **自動指令選單**：啟動後自動設定 Bot Menu，使用者可輕鬆查詢指令。
- **環境變數管理**：支援 `.env` 配置，保護機器人 Token 安全。
- **資源安全管理**：支援程式停止時自動安全存檔與關閉資料庫，防止資料毀損。

## 安裝與準備

1. **環境需求**：Python 3.9+
2. **安裝依賴**：

```bash
pip install -r requirements.txt
```

3. **配置參數**：
   在專案根目錄建立 `.env` 檔案並填入：

   ```env
   TG_BOT_TOKEN=你的機器人Token
   ```

   編輯 `main.py` 中的 `OWNER_ID`：

   ```python
   OWNER_ID = 0123456789  # 你的 Telegram ID
   ```

## 指令說明

### 使用者指令

- `/rank`：查詢個人等級報告（包含等級、XP 進度與視覺條）。
- `/top`：查看群組等級排行榜（支援分頁按鈕導覽）。

### 管理員指令 (需具備 Admin 權限)

- `/addxp @username 數量`：調整使用者 XP（支援負值。**亦可直接回覆某人訊息並輸入 `/addxp 數量`**）。
- `/addrank @username 等級數`：直接調整等級（支援負值。**亦可直接回覆某人訊息並輸入 `/addrank 數量`**）。
- `/addadmin @username/ID`：將使用者加入管理員名單。

### 擁有者指令 (僅限 OWNER_ID 使用)

- `/deladmin @username/ID`：將使用者從管理員名單移除。


## 資料庫結構 (SQLite)

機器人會自動建立 `chat_levels.db`，包含以下兩個資料表：

### `users` (用戶等級資料)

- `user_id`：Telegram 使用者唯一 ID
- `chat_id`：Telegram 群組/對話 ID
- `level`：當前等級
- `xp`：目前累積的經驗值
- `last_msg_time`：最後一次獲得 XP 的時間戳
- `username`：使用者帳號
- `display_name`：使用者顯示名稱（用於排行榜）

### `admins` (管理員名單)

- `user_id`：具備管理權限的使用者 ID

## 啟動

執行主程式：

```bash
python main.py
```

## 注意事項

- 本機器人預設處理所有非指令、非系統訊息的內容。
- **自動刪除**：為了保持群組整潔，大部分指令回應與升級訊息將在 60 秒後自動刪除。
- **多群隔離**：各群組的等級數據完全獨立，互不影響。
- 機器人必須在群組中具備「發送訊息」與「刪除訊息」的權限。
